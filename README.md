## **Program Descriptions**

### **1. AVL Tree Implementation**
- **File:** `AVL_TREE.cpp`
- **Description:** Implements an AVL Tree with insertion, deletion, and search operations. The tree maintains balance using rotations.

### **2. Consecutive Character Deletion using Stack**
- **File:** `ConsecutiveCharacterDeletion.cpp`
- **Description:** Deletes consecutive same characters in a sequence using a stack.

### **3. Binary Search Tree (BST) Implementation**
- **File:** `BST.cpp`
- **Description:** Implements a Binary Search Tree with insertion, deletion, and in-order traversal.

### **4. Bracket Reversal and Balance Check**
- **File:** `BracketReversalBalance.cpp`
- **Description:** Finds the minimum number of bracket reversals needed to make an expression balanced.

### **5. Book Stack Management System**
- **File:** `BookStackManagement.cpp`
- **Description:** Implements a stack-based book management system with push, pop, search, and delete operations.

### **6. Palindrome Check using Recursion**
- **File:** `Palindrome_recursive_check.cpp`
- **Description:** Checks if a given word is a palindrome using recursion.

### **7. Stack with Next Greater Element**
- **File:** `StackNextGreaterElement.cpp`
- **Description:** Implements a stack and finds the next greater element for each element in the stack.

### **8. Recursive Prime Number Check**

- **Description:** Checks if a number is prime using recursion.

### **9. Recursive Odd Number Display**

## **Installation Instructions**
1. **Clone the Repository:**
   ```bash
   git clone https://github.com/ihsansaif313/DSA.git
   ```
2. **Navigate to the repository directory:**
   ```bash
   cd DSA
   ```
3. **Compile and run any of the C++ files using a C++ compiler, for example:**
   ```bash
   g++ Palindrome_recursive_check.cpp -o Palindrome_recursive_check./Palindrome_recursive_check
   ```
