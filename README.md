## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/ihsansaif313/DSA.git
   cd DSA
   ```
2. **Compile & Run (Example: Palindrome Check)**
   ```bash
   g++ Palindrome_recursive_check.cpp -o palindrome
   ./palindrome
   ```
3. **Sample Output**
   ```bash
   Enter a word: racecar
   'racecar' is a palindrome!
   ```
